# Sound-Coder
Free audio steganography tool for MP3/AVI conversion.

## Features
 - MP3 to AVI encoding
 - AVI to WAV decoding
 - Simple GUI interface

## Özellikler
 - MP3'ten AVI'ye dönüşüm
 - AVI'den WAV'a çözümleme
 - Basit arayüz

## 特点
 - MP3转AVI编码
 - AVI转WAV解码
 - 简单界面

## Особенности
 - Конвертация MP3 в AVI
 - Декодирование AVI в WAV
 - Простой интерфейс
